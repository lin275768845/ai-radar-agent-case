# AI Radar Agent v0.2.0 Stable

This is the first stable trial-run version of AI Radar Agent.

Verified in this line:

- Feishu docx import as the primary reading surface.
- Feishu group card delivery.
- `brief.json` / `brief.md` generation.
- RSS / Bocha / Tavily provider audit.
- Markdown fallback when docx import fails or is unavailable.
- GitHub Actions manual and scheduled runs.

Known limits:

- Cross-run idempotency is not complete yet. `publish_result.json` can reuse a publish result inside an existing output directory, but GitHub Actions artifacts are temporary. Cross-run dedupe still needs GitHub cache, committed state, or Feishu-side lookup by document title.
- Tavily is not the default primary search source in this version. It is an optional quota-consuming supplement.
- Bocha Web Search is the recommended low-cost search replacement together with RSS.

Recommended default:

```text
SEARCH_PROVIDERS=rss,bocha
TAVILY_ENABLED=false
OUTPUT_MODE=feishu_docx_import
```
